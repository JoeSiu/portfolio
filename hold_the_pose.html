Hold the Pose!

an interactive WPF application where players have to match and hold their pose within a time limit

Turn on subtitle for description 

Details

Hold the Pose is an interactive WPF application where players have to match and hold their pose within a time limit. It also come with a pose editor where player can edit/ create new custom poses to make the game more interesting. It uses Kinect v2 for body tracking.

Pose editor features

  • Cycle through different poses and joint markers
  • Edit a pose by repositioning the joint markers
  • Delete a pose
  • Create a new pose using a custom image and place the joint markers onto it
  • Save all the poses into a .CSV file
  • Reset back to last saved states
  • Change the threshold for the joint detection
  • Go back to game view

Creation date: April, 2021
